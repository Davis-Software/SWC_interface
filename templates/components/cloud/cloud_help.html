Hotkeys

Hotkey Description
CTRL + Click Multiselect in cloud file list
CTRL + F Search files
ESC Exit from file search or deselect all files
RightClick File operations context menu for all selected files
CTRL + A Select all cloud files while hovering over the file list

Cloud good-to-knows

  • If you see a ">" before the file size of a directory, the directory contains more than 15 sub-folders and file size calculation has been interrupted to optimize performance
  • If the file size is listed as "error" and you receive a 404 error when trying to access the file, the file is protected or not readable in any way
